# Clase Git & GitHub - Laboratorio Colaborativo

Proyecto simple para practicar:

- `git clone`
- modificación de archivos
- `git status`
- `git add`
- `git commit`
- `git pull`
- `git push`
- revisión de cambios en GitHub

## Actividad

Cada alumno debe modificar una sola vista dentro de la carpeta alumnos.

## Comandos sugeridos

git clone URL_DEL_REPOSITORIO
cd clase-git-github-alumnos

git status
git add .
git commit -m "Alumno X modifica su vista"
git pull origin main
git push origin main
